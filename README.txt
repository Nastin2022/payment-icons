== Description ==

Add or change the default payment gateway icons that appear on the WooCommerce checkout page to anything you like or brand that suits your business.

== Installation ==

1. Upload `Payment_Gateway_Icon_For_WooCommerce folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Go to *WooCommerce* &rarr; *Settings* &rarr; *Payments* &rarr; select the payment gateway you want change its icon &rarr; below the settings of evrry payment method, you'll see additonal settings added below by this plugin. Enter an image URL to the *Icon* field.
